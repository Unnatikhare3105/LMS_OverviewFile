04 — REST API
Backend Endpoints
Fully typed, JWT-protected where required, with consistent error response schema across all routes.
Method Endpoint Description Auth
POST /api/user/register Create new user account Public
POST /api/user/login Auth & issue JWT token Public
GET /api/user/profile Fetch profile, stats, streaks JWT
GET /api/user/activity Heatmap activity data (time spent) JWT
GET /api/syllabus/search Search topics — text + video results JWT
GET /api/syllabus/:id Get full topic detail & content JWT
POST /api/quiz/generate AI-generate quiz for any topic JWT
POST /api/quiz/submit Submit answers, get score + rank update JWT
GET /api/bookmarks List user's saved content JWT
POST /api/bookmarks Add bookmark (text or video) JWT
DELETE /api/bookmarks/:id Remove bookmark JWT
GET /api/daily-challenge Today's challenge question JWT
